IT Integration Advisor/System Administrator

Advisicon, a professional services consulting company based in Vancouver, Washington seeks candidates for an IT Integration Advisor position.

Salary: $45000 to $55000 or higher depending on experience.

Summary

The IT Integration Advisor/System Administrator position entails the following client facing tasks, activities, and responsibilities:

  • Deliver SPS 2010/MOSS 2007 and SPF 2010/WSS 3.0 implementation consulting services to clients. This includes installation, migration, training, administration, and troubleshooting.
  • Deliver Project Server 2010/2007 implementation consulting services to clients. This includes installation, migration, training, administration, and troubleshooting.
  • Deliver basic IT Support to clients, including desktop, server, network, SQL, email and Microsoft Access application support and maintenance.
  • Consulting experience/Client facing/Training

The IT Integration Advisor/System Administrator position entails the following internal tasks, activities, and responsibilities:

  • Support end users and desktop systems.
  • Repairing old and deploying new computer systems.
  • Building computer images for deployment.
  • Provide system administration for internal systems:
    • SharePoint.
    • Project Server.
    • Exchange.
    • SQL Server.
    • CRM.
  • Manage technology infrastructure: Domain, network, and Windows Servers.
  • Manage system updates with Windows Server Update Services.
  • Maintain systems with System Center Operation Manager.
  • Support Virtual Servers in Hyper-V, Virtual Server, Virtual PC, or similar technology.
  • Manage Symantec Backup Exec system.
  • Maintain Websites and administer IIS.
  • Write technical Documentation for training manuals and internal IT processes
