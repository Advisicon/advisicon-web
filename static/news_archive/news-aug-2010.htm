Advisicon Project Management Training Now Available Online!

Advisicon will start offering online training courses this month. The initial courses, Practical Project Management (Best Practices) and PMP Exam Prep, will begin September 13th. The courses are structured in 2 hour sessions for 5 consecutive days.

Course Details  
  Duration: 5 days
  Start date: September 13th
  End date: September 17th

Course: Practical Project Management
  10:00AM to 12:00PM Pacific Time
  Participants will learn how to manage projects from initiation to close‐out while applying industry best practices. Students will also learn project management concepts, helpful standard terminology, and established methods based on the Project Management Institute (PMI®) Project Management Body of Knowledge (PMBOK®)

Course: PMP Exam Prep
  1:00PM to 3:00PM Pacific Time
  This course offers in-depth examination of the required study areas for PMI’s PMP exam. Participants will also learn tips and tricks for preparing for the exam. A study guide will be provided to participants.

For more information about these and other courses, visit Advisicon Training. To register for online courses, visit the Advisicon Course Registration portal. More classes coming soon!

Advisicon White Papers & Webcasts — Project Server 2010 and SharePoint Server 2010: Better Together

Advisicon recently released its third Microsoft White Paper and Webcast series— Project Server 2010 and SharePoint Server 2010: Better Together. The accompanying webcasts are Project Server 2010 and SharePoint Server 2010: Focus on Project Management and Project Server 2010 and SharePoint Server 2010: Focus on Collaboration.

The webcasts presented by Advisicon Senior Portfolio Advisor Tim Cermak and Advisicon Senior Project Advisor Manmeet Chaudhari highlight tools and integration features of SharePoint Server 2010 and Project Server 2010 that contribute to improved collaboration and communication in project management.

Previous white papers and webcasts in the series are Project Server 2010: A Look at Portfolio Strategy and Project Server 2010: A Look at Demand Management.
