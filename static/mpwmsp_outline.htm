Managing Projects with
Microsoft® Project

How to use Microsoft® Project to create
and manage project schedules.

TARGET AUDIENCE

DURATION
  • Project managers
  • Project team members
  • Project schedulers
  • Anyone who has used Microsoft® Project Professional and had a hard time getting it to do what they wanted
 2 Days

COURSE DESCRIPTION

This course teaches participants how to use Microsoft® Project to successfully create, track, manage, and modify project schedules and resource pools.  Participants will be introduced to industry best practices, tips and tricks for working with Microsoft® Project.  Our hands-on training is extremely practical and gives participants confidence to work with this very powerful, but often misunderstood and poorly used, project management tool.

LEARNING OBJECTIVES

TOOLS & TEMPLATES
  • Create, manage, and update effective project schedules
  • Create and manage resources, resource pools and assignments
  • Learn the do’s and don’ts of working with Microsoft® Project Professional
  • Create useful views for tracking, dashboards, printing, task usage, etc.
  • Formatting tricks for automating critical path tracking
  • Use reporting functions more effectively

 

Course participants receive a course manual and CD with tools and templates for key project management communications, analysis and reporting processes.
